Merhaba Ben Rana

En çok fasulye yemeğini severim. İngilizce ve Almanca öğreniyorum. Favori rengim kırmızıdır. Kısaca yeni şeyler öğrenmeyi seviyorum.

Scratch ile 30'dan fazla oyun ve uygulama geliştirdim.

HTML ve CSS ile YouTube videoları yardımı ile kodlama yapmaya devam ediyorum. Github hesabımda birkaç projem var.

Python öğrenmeye başladım. Jupyter notebookta basit hesap makinesi kodladım.
